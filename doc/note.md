# Spring AOP

| 概念          | 含义                                         |
| ------------- | -------------------------------------------- |
| Aspect        | 切面                                         |
| Join Point    | 连接点，Spring AOP里总是代表一次方法执行     |
| Advice        | 通知，在连接点执行的动作                     |
| Pointcut      | 切入点，说明如何匹配连接点                   |
| Introduction  | 引入，为现有类型声明额外的方法和属性         |
| Target object | 目标对象                                     |
| AOP proxy     | APO代理对象，JDK动态代理/GCLIB代理           |
| Weaving       | 织入，连接切面与目标对象或类型创建代理的过程 |
